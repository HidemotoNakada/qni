Usage

The <quantum-circuit> component is a custom web component to display a quantum circuit defined in JSON. To pass quantum circuit JSON data to the <quantum-circuit> component, you can either specify it directly in the HTML as a data-json attribute, or you can set the data-json attribute with JavaScript.

The Qni quantum circuit JSON format is a representation of each step in the circuit as an array. For example, in a three-qubit circuit, the step to apply Hadamard gates to each qubit is represented as ["H","H","H"]. By concatenating multiple steps defined in this way, an array defining the quantum circuit can be created. The Qni quantum circuit JSON format holds this array as the value of the JSON key "cols".

Embedding JSON

<quantum-circuit data-json="{&quot;cols&quot;:[[&quot;|0>&quot;,&quot;|0>&quot;],[&quot;H&quot;,1],[&quot;â€¢&quot;,&quot;X&quot;],[&quot;Measure&quot;,1],[1,&quot;Measure&quot;]]}"></quantum-circuit>

Set JSON with JavaScript

<quantum-circuit id="quantum-circuit"></quantum-circuit>

<script>
  document.getElementById("quantum-circuit").setAttribute("data-json", '{"cols":[["|0>","|0>"],["H",1],["â€¢","H"],["Measure",1],[1,"Measure"]]}')
</script>

Conditional Flag

If you want to control the application of a quantum gate by a flag, you can create a flag with the data-flag attribute. If the value of the measurement gate is 1, this flag will be on; if it is 0, it will be off. The quantum gate with this flag will be applied only if the flag is on. 

<measurement-gate data-flag="foobar"></measurement-gate>

User Help Popup

As a user help, you can display the description and usage of the Measurement gate as a popup. The popup will show on the mouse hover event. The content to be displayed can be specified in the <measurement-gate> tag as follows:

Measurement Gate

Measures whether a qubit is ON or OFF.

 
<measurement-gate>
  <header>
    <h1>Measurement Gate</h1>
    <p>Measures whether a qubit is ON or OFF.</p>
  </header>
</measurement-gate>

Draggable

To make the Measurement gate drag-and-droppable, specify the data-draggable attribute. 

<measurement-gate data-draggable><measurement-gate>
