Rz Gate

Rotates around the Z-axis.

Attribute
State
data-size="xs" height: 1rem; width: 1rem;
data-size="sm" height: 1.5rem; width: 1.5rem;
data-size="base" height: 2rem; width: 2rem;
data-size="lg" height: 2.5rem; width: 2.5rem;
data-size="xl" height: 3rem; width: 3rem;
data-theta="value" θ = value
data-disabled Disabled
data-wire-top Connedted with lower bit gate
data-wire-bottom Connected with upper bit gate
data-wire-top-disabled Connected with lower bit gate (disabled)
data-wire-bottom-disabled Connected with upper bit gate (disabled)

Sizes

Control the size of a Rz gate using the data-size attribute. 

<rz-gate data-size="xs"></rz-gate>
<rz-gate data-size="sm"></rz-gate>
<rz-gate data-size="base"></rz-gate>
<rz-gate data-size="lg"></rz-gate>
<rz-gate data-size="xl"></rz-gate>

Angle

Control the angle θ of a Rz gate using the data-theta attribute. 

<rz-gate data-theta="0.123"></rz-gate>

Disabled Gate

To display the disabled Rz gate, set the data-disabled attribute. 

<rz-gate data-disabled></rz-gate>

Connecting with Other Gates

To connect the Rz gate to the quantum gate of the lower bit, set the data-wire-top attribute. Similarly, set data-wire-bottom for the upper bits. 

<rz-gate data-wire-top></rz-gate>
<rz-gate data-wire-bottom></rz-gate>
<rz-gate data-wire-top data-wire-bottom></rz-gate>
<rz-gate data-wire-top-disabled></rz-gate>
<rz-gate data-wire-bottom-disabled></rz-gate>
<rz-gate data-wire-top-disabled data-wire-bottom-disabled></rz-gate>

To show disabled connections, use the data-wire-top-disabled and data-wire-bottom-disabled attributes. 

<rz-gate data-wire-top-disabled></rz-gate>
<rz-gate data-wire-bottom-disabled></rz-gate>
<rz-gate data-wire-top-disabled data-wire-bottom-disabled></rz-gate>
<rz-gate data-wire-top-disabled data-disabled></rz-gate>
<rz-gate data-wire-bottom-disabled data-disabled></rz-gate>
<rz-gate data-wire-top-disabled data-wire-bottom-disabled data-disabled></rz-gate>
